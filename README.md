Cybersecurity Chatbot

This is a console-based chatbot that teaches users about cybersecurity topics 
like phishing, malware, VPNs, and safe browsing. The chatbot interacts with 
users and responds based on their input.

Features
- Responds to cybersecurity questions
- Detects user sentiment (worried, curious, frustrated)
- Plays a greeting audio on startup
- ASCII art welcome screen
- Exit command to close the chatbot

Requirements
- .NET (version you used, e.g. .NET 6 or 8)
- Visual Studio

How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run the program
4. Type messages into the console to interact with the chatbot
5. Type "exit" to close the program

Example:
Buck: Howdy, welcome to the Cybersecurity Chatbot!
You: what is phishing
Buck: Phishing's like a snake oil salesman, tryin' to swindle your secrets outta ya.

Project Structure
- Chatbot.cs → main chatbot logic
- AudioPlayer.cs → handles sound playback
- User.cs → handles user input
- greeting.wav → startup audio
